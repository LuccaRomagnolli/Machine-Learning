# Machine-Learning

Repositório com notebooks e materiais de aprendizado de máquina, focado em exemplos práticos.

**Autor:** Lucca Romagnolli

**Conteúdo principal:**
- `Notebooks/` — colecção de códigos e projetos:
  - `Binnary-Classification.ipynb` — exemplos de classificação binária
  - `Logistic_Regression.ipynb` — regressão logística aplicada
  - `K-Means-Classifier.ipynb` — clustering com K-Means
  - `Ensemble-Models.ipynb` — implementação e comparação de ensembles
  - ...
- `Materiais.md` — anotações e referências para estudantes.

**Como usar (rápido):**
1. Recomenda-se usar um ambiente virtual com Python 3.8+.
2. Instale dependências (se houver `requirements.txt`):

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

3. Abra os notebooks com `jupyter lab` ou diretamente no VS Code.
